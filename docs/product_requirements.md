# 产品需求说明

## 产品定位

公司决策监控是一款面向公司研究、投资研究、企业动态跟踪的桌面端信息监控工具。当前版本为 UI Preview Mode，只展示正式产品的信息架构、页面结构和交互占位。

## 目标用户

- 公司研究员
- 投资研究员
- 企业战略与竞品分析人员
- 需要跟踪公司公告、新闻和风险信号的桌面端用户

## 核心场景

1. 搜索公司并进入公司详情模板。
2. 将公司加入自选列表，集中跟踪动态。
3. 查看热门公司、风险监控和 AI 总结模块的产品入口。
4. 在设置页理解当前版本边界和未来数据源接入方式。

## 核心页面

- Dashboard 首页
- Search 公司搜索
- Company Detail 公司详情
- Watchlist 自选公司
- Hot Companies 热门公司
- Risk Monitor 风险监控
- AI Summary AI 总结
- Settings 设置

## 当前版本边界

- 不包含真实公司数据。
- 不接入真实 API。
- 不执行真实联网搜索。
- 不执行真实爬虫。
- 不写入真实业务数据库。
- 不提供投资建议、交易建议或买卖功能。
